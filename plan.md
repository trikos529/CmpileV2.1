# Python Project
# Project Name: Cmpile
# Project Description: Cmpile is a Python project that allows users to compile and run C and C++ code.
# Project Version: 2.0.0

# Project Structure
# cmpile/
# ├── cmpile.py
# ├── download_script.py
# ├── vcpkg_automation.py
# ├── package_finder.py
# ├── ui.py
# internal_downloads/
# ├── vcpkg
# ├── gcc

# Instructions
# cmpile.py: The script that runs the program.
# download_script.py: The script that downloads the gcc compiler and vcpkg into the internal downloads folder.
# vcpkg_automation.py: The script that automates the package installation and linking via vcpkg.
# package_finder.py: The script that finds the required packages in the c/cpp codebase.
# ui.py: The script that creates the user interface.

# Internal Downloads
# vcpkg: The vcpkg package manager.
# gcc: The gcc compiler.
